Регулярные выражения JavaScript.

  • - Регулярное выражение для проверки корректности e-mail:

    ^([a-z0-9_-]+\.)*[a-z0-9_-]+@[a-z0-9_-]+(\.[a-z0-9_-]+)*\.[a-z]{2,6}$

  • - Регулярное выражение для проверки корректности URL:

    ^((https?|ftp)\:\/\/)?([a-z0-9]{1})((\.[a-z0-9-])|([a-z0-9-]))*\.([a-z]{2,6})(\/?)$

  • - Регулярное выражение для проверки корректности номера телефона (пример: «+78(555)555-55-55»):

    ^\+\d{2}\(\d{3}\)\d{3}-\d{2}-\d{2}$

  • - Пример использования:

    function isValidPhone(myPhone) { return /^\+\d{2}\(\d{3}\)\d{3}-\d{2}-\d{2}$/.test(myPhone); }
