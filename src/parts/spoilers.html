Динамические спойлеры.

  • - Для родителя спойлеров пишем атрибут:

    data-spoilers

  • - Для заголовка спойлеров пишем атрибут:

    type="button" tabindex="-1" data-spoiler

  • - Для спойлеров типа "аккордеон" пишем атрибут:

    data-one-spoiler

  • - Для динамических спойлеров атрибуту data-spoilers задаем параметры ширины экрана и тип breakpoint, например:

    data-spoilers = "992,min" data-spoilers = "768,max"
